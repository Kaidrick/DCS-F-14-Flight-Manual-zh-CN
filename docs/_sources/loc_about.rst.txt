关于本项目
##############


本文档是\ **DCS: F-14 “雄猫” 飞行手册**\ 的官方中文本地化项目。

文档中的图片和文字资料均来自\ `官方在线手册 <http://www.heatblur.se/F-14Manual/>`_。

随着官方手册的不断更新和完善，文档中的内容会有变动。我们呼吁任何有能力和有热情的玩家为这个开源本地化项目出一份力。
Github 页会尽可能频繁地更新文档源并添加翻译，提供的所有相关文件（包括图片的 Photoshop 工程文件）可用于第三方网站架设或作为线下文档保存。

出于各种各样的考量（如，技术原因、原文档包含*.gif文件），暂时不提供 LaTeX 或 PDF 文档。如有需要，请自行修改文档，并通过 sphinx-doc 导出 LaTeX。
如需更多帮助，请在 Github 联系项目管理人员。

上述的各类项目文件，以及关于 reStructuredText 和 sphinx-doc 的基本使用方法，以及翻译中需要注意的细节，请查阅
\ `本项目 Github 页面 <https://github.com/Kaidrick/DCS-F-14-Flight-Manual-zh-CN/>`_\ 。
